Diagram 2 — Multi-agent pipeline (the core LangGraph execution):

flowchart TD
    START([START]) --> SUP

    SUP["Supervisor Agent
    — understands intent
    — returns JSON plan"]

    SUP --> MA
    MA --> FA
    FA --> KA
    KA --> STA

    MA["Market Agent  gpt-4o-mini
    tools: market size, competitors,
    regulatory, trends
    output → market_insights"]

    FA["Financial Agent  gpt-4o-mini
    tools: macro indicators, ROI,
    credit risk
    output → financial_analysis"]

    KA["Knowledge Agent  gpt-4o-mini
    tools: company profile, past
    expansions, risk policies
    output → knowledge_summary"]

    STA["Strategy Agent  gpt-4o
    pure LLM synthesis
    scores all 3 inputs
    output → GO / NO-GO decision"]

    STA --> CA

    CA["Communication Agent  gpt-4o
    generates markdown
    executive report
    output → final_report"]

    CA --> END([END])

    style MA fill:#E6F1FB,stroke:#185FA5,color:#0C447C
    style FA fill:#E6F1FB,stroke:#185FA5,color:#0C447C
    style KA fill:#E6F1FB,stroke:#185FA5,color:#0C447C
    style STA fill:#EEEDFE,stroke:#534AB7,color:#3C3489
    style CA fill:#EEEDFE,stroke:#534AB7,color:#3C3489
